# Word Pad

A lightweight text editor written in **C** using **Raylib** for rendering and **Raygui** for the user interface.

This project is being built from scratch as a learning project to understand how text editors work internally instead of relying on existing editor widgets.

---

## Features

### Current

- Text input
- Multi-line editing
- Automatic line wrapping
- Cursor rendering
- Real-time text rendering

### Planned

- Backspace
- Arrow key navigation
- File menu
- Open/Save files
- Undo / Redo
- Copy / Paste
- Text selection
- Scrolling
- Status bar
- Keyboard shortcuts
- Themes
- Syntax highlighting (future)
